# xuichart
基于echarts的 基于特定业务的二次封装，为满足业务开发场景中一些标准的，通用的图表展示效果，封装了基于echarts和图表组件库，
主要是能简化一些简单图表的echarts的开发，减少代码量和代码可控，能对通用的一些配置在主题中调整，也可以在xuichart封装代码里调整
